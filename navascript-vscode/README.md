# 🐺 NavaScript Support for VS Code

¡Bienvenido a la extensión oficial de **NavaScript (NS)**! Esta herramienta transforma Visual Studio Code en un entorno de desarrollo completo para programar con la sintaxis de la manada.

---

## ✨ Características

- **Resaltado de Sintaxis:** Colores distintivos para palabras clave como `lobo aparece`, `SHAREEE`, `Gii Huu` y más.
- **Ejecutor Integrado (RUNRUN):** Ejecuta tu código directamente desde el editor con un solo clic.
- **Consola Dedicada:** Salida de datos limpia en el panel de "NavaScript Console".
- **Comentarios con Estilo:** Soporte completo para comentarios tipo `Rawr`.

---

## 🚀 Cómo empezar

1. Crea un nuevo archivo con la extensión `.ns` (ejemplo: `inicio.ns`).
2. Escribe tu primer código:

```ns
Rawr Mi primer programa
lobo aparece saludo = "¡Hola Manada!"
SHAREEE saludo
```

3. Haz clic en el icono de Play (**▶️**) en la esquina superior derecha del editor, o presiona `Ctrl+Shift+P` y busca `NS: Ejecutar RUNRUN`.

---

## 📖 Guía rápida de sintaxis

| Palabra Clave            | Función                          |
|--------------------------|----------------------------------|
| `lobo aparece x = 5`     | Declarar una variable            |
| `SHAREEE "texto"`        | Imprimir en la consola           |
| `Gii Huu fn() { }`       | Definir una función              |
| `si cond { } sino { }`   | Condicionales                    |
| `mientras cond { }`      | Bucle while                      |
| `Grr cond { }`           | Bucle "hasta que" (while invertido) |
| `Zap x`                  | Incrementar variable             |
| `Boom x`                 | Decrementar variable             |
| `Uuh 1000`               | Pausa en milisegundos            |
| `Rawr`                   | Comentario de línea              |

---

## 🛠️ Requisitos
